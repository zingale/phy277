2. Consider the following---what do you think the output printed
   to the screen is?

   .. code:: c++

      #include <vector>
      #include <algorithm>
      #include <iostream>

      bool is_even(int e) {
          return e % 2 == 0;
      }

      int main() {

          std::vector<int> vec{1, 2, 3, 4, 5, 6, 7, 8};
          auto val = std::ranges::count_if(vec, is_even);

          std::cout << val << std::endl;
      }

   .. dropdown:: solution
      :color: muted
      :icon: pencil
      :animate: fade-in-slide-down

      This will only count the elements in the vector for which
      ``is_even`` returns ``true``, so we will get ``4``.
