**********
matplotlib
**********

`matplotlib <https://matplotlib.org/>`_ is the core visualization package in python.
We import it as:

.. code:: python

   import matplotlib.pyplot as plt

This puts all of the main plotting functions / classes into the ``plt`` namespace.
